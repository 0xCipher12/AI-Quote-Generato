About Wisdom Arc

The Vision

Wisdom Arc is designed to inspire users through meaningful and motivational quotes. It combines simplicity with creativity, allowing users to explore ideas and express their own thoughts.

Key Features

  • Generate inspirational quotes instantly
  • Write your own personal quotes
  • Save quotes using browser storage
  • Delete and manage saved quotes
  • Clean and modern user interface

How It Works

The application uses JavaScript to dynamically generate quotes and manage user input. Saved quotes are stored in the browser using LocalStorage, allowing users to revisit their ideas anytime.

Technologies Used

HTML5 CSS3 JavaScript LocalStorage

Project Details

Version: 1.1.0
Type: Frontend Web Application
Storage: Browser LocalStorage
Purpose: Academic Project & Skill Demonstration
