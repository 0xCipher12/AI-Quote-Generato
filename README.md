# ✨ Wisdom Arc – AI Quote Generator

Wisdom Arc is a modern, minimalistic quote generator web application that delivers unique and inspiring quotes with a clean UI.

---

## 🚀 Features

- 🎯 Generate random inspirational quotes instantly  
- ⚡ Fast and responsive UI  
- 🎨 Clean and modern design  
- 🔄 Dynamic content using API  
- 💡 Beginner-friendly project with real-world concepts  

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **Public Quote API**
