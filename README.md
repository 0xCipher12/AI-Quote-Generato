# ✨ Wisdom Arc – AI Quote Generator

Wisdom Arc is a modern and minimalistic quote generator web application that delivers unique and inspiring quotes with a clean and user-friendly interface.

It not only generates quotes but also allows users to **write, save, and manage their own quotes**, making it more interactive and personalized.

---

## 🚀 Features

* 🎯 Generate random inspirational quotes instantly
* ✍️ Write your own custom quotes
* 💾 Save quotes using browser LocalStorage
* ❌ Delete saved quotes
* 📊 View saved quotes count (homepage)
* ⚡ Fast and responsive UI
* 🎨 Clean and modern design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* LocalStorage
